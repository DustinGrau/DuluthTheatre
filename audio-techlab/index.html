Audio Tech Lab

Behringer Europower PMP3000

This is an usual combination of both audio mixing surface and 2-channel amplifier allowing for a compact installation that does not depend on either a standalone amplifier or a self-powered speaker. The system is entirely analog and represents the most basic functionality you may find on any modern audio mixer. Understanding the basics of signal routing and effects on output will prepare you to find your way when using any modern or digital mixing surface.

Inputs

Unless distinctly noted, consider every input of a mixer to accept a monoaural (or mono) input. This is a single-channel audio source which will be played back equally in all speakers, so long as the pan/panning dial is set to the center position.

Wired Microphone

Channel 1 uses an XLR connection for balanced input and represents a simple, directly-connected microphone.

Wireless Microphone

Channel 2 uses an XLR connection for balanced input and represents a commmon wireless microphone configuration to a receiver.

Line Input

Channel 8 uses a line level input (TS or TRS 1/4" phone/phono connector) to directly connect a laptop or other audio playback device. This connection is clearly labelled as accepting either a balanced or unbalanced signal.

Outputs

This mixer has 2 unique outputs which are powered, and are labelled as A and B on the back of the console and utilize Speakon (twist-lock) connectors. These output are purposefully not labelled as simply "L" (left) or "R" (right) as you might expect, as their purpose can be configured.

WARNING: These must ONLY be connected to a passive speaker!

A dedicated "AMP MODE" switch can be found at the top-right of the mixing surface, as a white box with "A" and "B", and can be set to 1 of 3 possible positions. For use with a single speaker the switch should be set to the middle position which has a "MON1" label on the "A" side of the switch and "MONO" label on the "B" side of the switch. This will send the MON1 mix to output A and the MONO mix to output B. In this configuration any stereo inputs will be output only as a monoaural output as there is only 1 speaker available for this lab setup. In this configuration the "program" or main output will be controlled by the MONO fader on the far right-hand side of the mixing surface.

Other Mixers

Other mixing consoles of various models and conditions can be found in the storage space under the stairs leading to the costume loft. These may or may not be usable for any purposes but do exist if needed for learning or as props.
